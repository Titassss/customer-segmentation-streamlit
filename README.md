# Customer Segmentation & Profiling App

### This project implements a customer segmentation system using unsupervised learning and deploys it as an interactive Streamlit application.

## Project Overview

- Segments customers using KMeans clustering

- Translates clusters into interpretable customer personas

- Provides a dashboard for cluster-level analysis and comparison

- Enables individual customer profiling with real-time cluster assignment

## Technical Details

- Scikit-learn for data preprocessing and KMeans clustering

- Pandas for data loading and feature handling

- Streamlit for building the interactive dashboard and inference interface
